Integrate a Caldera Form with Slack and get submissions notification to a Slack Channel.

== Description ==

Set up a support, contact or a simple "Hi" form in the free <a href="https://wordpress.org/plugins/caldera-forms/">Caldera Forms</a> and connect it to Slack channel to get instant notifications on form submissions.

[youtube https://www.youtube.com/watch?v=Y8TjvQvctGw]

Slack Integration for Caldera Forms is a free plugin by [CalderaWP](https://CalderaWP.com).

== Installation ==

Install the plugin through the WordPress Plugins Installer or upload `plugin-name.php` to the plugins directory of your content directory
1. Activate the plugin through the 'Plugins' menu in WordPress
Create a form or Edit an existing Form.
Click on the "Processors" Tab.
Click Add Processor and select "Slack"
Setup your webhook URL and various fields, save and presto!
